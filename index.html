Project Overview

This project explores user churn within the Waze application using a complete end-to-end Machine learning pipeline. It includes data preprocessing, exploratory data analysis, model building, and deployment strategy.

Explore on GitHub

Gallery

Throughout the analysis, various visualizations were generated using libraries like Seaborn and Matplotlib to communicate patterns in user behavior and model performance.

Models & Evaluation

We tested several models including Logistic Regression, Random Forest and XGBoost Evaluation metrics included accuracy, precision, recall, and F1-score.
