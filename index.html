Waze Churn Prediction

Predicting user churn with machine learning and behavioral insights

View on GitHub

Project Overview

This project aims to predict user churn for the Waze navigation app using machine learning models. The goal is to identify users likely to leave the platform and provide actionable insights for retention strategies.

Dataset & Analysis

The dataset includes anonymized user behavior data from Waze, including metrics like trip frequency, support interactions, and user engagement. Key features were explored to identify patterns related to churn behavior.

Machine Learning Models

We used various machine learning models for churn prediction, including Random Forest, XGBoost, and SVM. Each model was evaluated based on ROC-AUC and other relevant metrics.
