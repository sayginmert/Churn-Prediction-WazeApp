Waze Churn Prediction

Predicting user churn using Machine Learning

View on GitHub

Project Overview

This project explores user churn within the Waze application using a complete end-to-end machine learning pipeline. It includes data preprocessing, exploratory data analysis, model building, and deployment strategy.

Dataset & Analysis

The data used was synthetic but representative of real-world Waze usage. We conducted exploratory data analysis to identify key churn indicators.

Models & Evaluation

We tested several models including Logistic Regression, Random Forest, XGBoost, and SVM. Evaluation metrics included accuracy, precision, recall, and F1-score.

Visualizations

Throughout the analysis, various visualizations were generated using libraries like Seaborn and Matplotlib to communicate patterns in user behavior and model performance.
